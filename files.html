Text Processing & File Hunting (grep, awk, cut, etc.)

grep (Filtering)

Basic search:

grep "string" file.txt

Case insensitive:

grep -i "admin" file.txt

Show line numbers:

grep -n "error" file.txt

Regex:

grep -E "^[0-9]" file.txt

Exclude lines:

grep -v "string" file.txt
awk (Column Extraction)

Print specific column:

awk '{print $1}' file.txt

Multiple columns:

awk '{print $1, $3}' file.txt

Filter + print:

awk '$1 ~ /^[0-9]/ {print $2}' file.txt

Change delimiter:

awk -F ':' '{print $1}' file.txt
cut (Field Extraction)

Delimiter based:

cut -d ':' -f1 file.txt

Multiple fields:

cut -d ':' -f1,3 file.txt
sort / uniq (Cleaning Data)
sort file.txt
sort -u file.txt
uniq file.txt

Count occurrences:

sort file.txt | uniq -c
tr (Transform Text)

Lowercase:

tr 'A-Z' 'a-z' < file.txt

Remove characters:

tr -d ':' < file.txt
sed (Stream Editing)

Replace text:

sed 's/old/new/g' file.txt

Delete lines:

sed '/pattern/d' file.txt
File Hunting (find, locate, strings, etc.)

Find files:

find / -name "*.txt" 2>/dev/null
find / -type f -name "*.conf" 2>/dev/null
find / -perm -4000 2>/dev/null
find / -writable -type f 2>/dev/null

Fast search:

locate passwd

Search inside files:

grep -R "password" .
rg "password" .

Binary analysis:

strings file.bin | grep -i password

Identify file type:

file unknown.bin
Real OSCP Pipelines (Very Important)

Extract ports from nmap output:

grep "^[0-9]" openPorts | awk '{print $1}' | cut -d'/' -f1 | paste -sd,

Get IPs from file:

grep -oE '[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+' file.txt | sort -u

Clean usernames list:

cut -d ':' -f1 users.txt | sort -u

Filter valid results:

cat file.txt | grep -i "open" | awk '{print $2}'

Extract emails:

grep -oE '[a-zA-Z0-9._%+-]+@[a-zA-Z0-9.-]+' file.txt
OSCP Tips
  • Always think in pipelines: grep | awk | cut | sort
  • Use grep -o to extract matches only
  • Use sort -u to remove duplicates
  • Use paste -sd, to format lists (ports, IPs)
  • Combine find + grep for deep searches
  • Prefer rg for large directories when available
