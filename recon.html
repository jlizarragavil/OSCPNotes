Recon & Enumeration Cheatsheet (OSCP Style)

1. Nmap Quick Scan (First Look)
sudo nmap -p- --open --min-rate 5000 -sS -n -Pn target_ip -oN openPorts

What you do:

  • Quickly identify open ports
  • Detect service versions

OSCP Tip: Run this first and continue working while it executes.

2. Full TCP Scan (All Ports)
nmap -p- -T4 -Pn -n target_ip -oN full_ports.txt
nmap -sCV -p $(grep "^[0-9]" openPorts | awk '{print $1}' | cut -d'/' -f1 | paste -sd,) target_ip -oN services
3. UDP Scan (If Time Allows)
nmap -sU --top-ports 50 target_ip
4. Service Enumeration Mindset
  • What service is this?
  • What version?
  • Is authentication required?
Enumerate → Find → Exploit → Repeat
5. Port 21 - FTP
ftp target_ip
anonymous:anonymous
6. Port 22 - SSH
ssh user@target_ip
7. Port 80/443 - Web
whatweb http://target_ip
nikto -h http://target_ip
gobuster dir -u http://target_ip -w wordlist.txt -x php,txt,html
8. Port 139/445 - SMB
smbclient -L //target_ip -N
enum4linux-ng target_ip
smbmap -H target_ip
smbclient //target_ip/share -N -c "recurse ON; prompt OFF; ls"
rpcclient -U "" target_ip

Check: shares, creds, writable folders

9. Port 53 - DNS
dig axfr @target_ip domain.local
10. Port 389 - LDAP
ldapsearch -x -h target_ip -b "dc=domain,dc=local"
11. Port 3389 - RDP
xfreerdp /u:user /p:password /v:target_ip
12. Web Enumeration Deep
ffuf -u http://target_ip/FUZZ -w wordlist.txt
  • robots.txt
  • view-source
  • JS files
  • hidden endpoints
13. Credential Hunting
grep -R "password" .
find / -name "*.conf" 2>/dev/null
14. Linux PrivEsc
sudo -l
find / -perm -4000 2>/dev/null
linpeas.sh
15. Windows PrivEsc
whoami /priv
systeminfo
net user
winpeas.exe
16. Active Directory Basics
rpcclient -U "" target_ip
  • Users
  • Groups
  • Shares
17. OSCP Attack Loop
  1. Full port scan
  2. Service enumeration
  3. Web + SMB first
  4. Search exploits
  5. Try creds everywhere
  6. Repeat loop
When stuck → change vector, not machine
