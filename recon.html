Recon & Enumeration Cheatsheet (OSCP Style)

1. Nmap Quick Scan (First Look)
sudo nmap -p- --open --min-rate 5000 -sS -n -Pn target_ip -oN openPorts

What you do:

  • Quickly identify open ports
  • Detect service versions
  • Run default scripts

OSCP Tip: Run this first and continue working while it executes.

2. Full TCP Scan (All Ports)
nmap -p- -T4 -Pn -n target_ip -oN full_ports.txt

Then:

nmap -sCV -p $(grep "^[0-9]" openPorts | awk '{print $1}' | cut -d'/' -f1 | paste -sd,) IP -oN services

What you are looking for:

  • Uncommon ports
  • Hidden services
3. UDP Scan (If Time Allows)
nmap -sU --top-ports 50 target_ip

Important:

  • Slow → run in background
  • Look for DNS, SNMP, NFS
4. Service Enumeration Mindset

Do not think in terms of attacks → think:

  • What service is this?
  • What version?
  • Is authentication required?
  • What data is exposed?
Enumerate → Find → Exploit → Repeat
5. Port 21 - FTP
ftp target_ip
anonymous:anonymous
nmap --script ftp-* -p 21 target_ip

What to check:

  • Anonymous login
  • Writable directories
  • Interesting files
6. Port 22 - SSH
ssh user@target_ip
nmap --script ssh-* -p 22 target_ip

What to check:

  • Valid credentials
  • Vulnerable versions
7. Port 80/443 - Web
whatweb http://target_ip
nikto -h http://target_ip
gobuster dir -u http://target_ip -w wordlist.txt

What to check:

  • Hidden directories
  • Login panels
  • Upload functionality
8. Port 139/445 - SMB
smbclient -L //target_ip -N
enum4linux -a target_ip
crackmapexec smb target_ip --shares

What to check:

  • Accessible shares
  • Usernames
  • Credentials
9. Port 53 - DNS
dig axfr @target_ip domain.local
dnsrecon -d domain.local

What to check:

  • Zone transfers
  • Subdomains
10. Port 389 - LDAP
ldapsearch -x -h target_ip -s base
ldapsearch -x -h target_ip -b "dc=domain,dc=local"

What to check:

  • Users
  • Domain structure
11. Port 3389 - RDP
xfreerdp /u:user /p:password /v:target_ip

What to check:

  • Valid login
  • Credential reuse
OSCP Mental Flow
  1. Run quick scan (nmap -sC -sV)
  2. Run full port scan
  3. Enumerate each service
  4. Focus on web if available
  5. Check SMB / FTP
  6. Try credentials everywhere
  7. Repeat the loop
