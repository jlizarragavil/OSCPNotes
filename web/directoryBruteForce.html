Gobuster

Directory brute force

gobuster dir -u <IP> -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -t 50

Basic fuzzing

gobuster dir -u http://<IP>/ -w <WORDLIST> -c -t 50

Search for files with extensions

gobuster dir -u http://<IP>/ -w <WORDLIST> -x php,txt,html -c -t 50

Filtering (status codes)

gobuster dir -u http://<IP>/ -w <WORDLIST> -s 200,301,302 -c

Fuzzing parameters

gobuster dir -u "http://<IP>/<SUBDIRECTORY>/?FUZZ=1" -w <WORDLIST> -c

Fuzzing auth pages (POST)

gobuster dir -u http://<IP>/<SUBDIRECTORY>/ -w <WORDLIST> -m POST \
--data 'uname=Dummy&passwd=FUZZ&submit=Submit' \
-H 'Content-Type: application/x-www-form-urlencoded' -t 50

Subdomain enumeration

gobuster vhost -u http://FUZZ.mydomain.com -w <WORDLIST>
gobuster vhost -u http://mydomain.com -w <WORDLIST> --append-domain

Virtual hosting

gobuster vhost -u http://mydomain.com -w <WORDLIST> -H 'Host: FUZZ.mydomain.com'

Virtual hosting with IP

gobuster vhost -u http://<IP> -w <WORDLIST> -H 'Host: FUZZ.thm'
Feroxbuster

Full scan example

feroxbuster -u http://192.168.45.242:28080 -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -t 100 -d 1

Basic directory fuzzing

feroxbuster -u http://<IP>/FUZZ -w <WORDLIST> -c -t 50

Search for files

feroxbuster -u http://<IP>/FUZZ -w <WORDLIST> -e php,txt -c

Filtering

feroxbuster -u http://<IP>/FUZZ -w <WORDLIST> --filter-status 403 -c

Fuzzing parameters

feroxbuster -u 'http://<IP>/<SUBDIRECTORY>/?FUZZ=1' -w <WORDLIST> --filter-size 39 -c

Fuzzing auth pages (POST)

feroxbuster -u http://<IP>/<SUBDIRECTORY>/ -X POST \
-d 'uname=Dummy&passwd=FUZZ&submit=Submit' \
-H 'Content-Type: application/x-www-form-urlencoded' \
--filter-size 1435 -c

Subdomain enumeration

feroxbuster -u http://FUZZ.mydomain.com -w <WORDLIST> --filter-size 0 -c

Virtual hosting

feroxbuster -u http://mydomain.com -w <WORDLIST> -H 'Host: FUZZ.mydomain.com' --filter-size 0 -c

Virtual hosting with IP

feroxbuster -u http://<IP> -w <WORDLIST> -H 'Host: FUZZ.thm' -c
ffuf

Basic fuzzing

ffuf -c -u http://<IP>/FUZZ -w <WORDLIST> -t 50

Search for files

ffuf -c -u http://<IP>/FUZZ -w <WORDLIST> -e .php,.txt

Filtering

ffuf -c -u http://<IP>/FUZZ -w <WORDLIST> -fc 403

Fuzzing parameters

ffuf -u 'http://<IP>/<SUBDIRECTORY>/?FUZZ=1' -c -w <WORDLIST> -fw 39

Fuzzing auth pages (POST)

ffuf -u http://<IP>/<SUBDIRECTORY>/ -c -w <WORDLIST> -X POST \
-d 'uname=Dummy&passwd=FUZZ&submit=Submit' \
-H 'Content-Type: application/x-www-form-urlencoded' -fs 1435

Subdomain enumeration

ffuf -u http://FUZZ.mydomain.com -c -w <WORDLIST> -fs 0

Virtual hosting

ffuf -u http://mydomain.com -c -w <WORDLIST> -H 'Host: FUZZ.mydomain.com' -fs 0

Virtual hosting with IP

ffuf -u http://<IP> -H 'Host: FUZZ.thm' -w <WORDLIST>
wfuzz

Basic fuzzing

wfuzz -c -w <WORDLIST> http://<IP>/FUZZ

Search for files

wfuzz -c -w <WORDLIST> -z list,.php,.txt http://<IP>/FUZZ

Filtering (exclude 403)

wfuzz -c -w <WORDLIST> --hc 403 http://<IP>/FUZZ

Fuzzing parameters

wfuzz -c -w <WORDLIST> --hc 39 http://<IP>/<SUBDIRECTORY>/?FUZZ=1

Fuzzing auth pages (POST)

wfuzz -c -w <WORDLIST> -X POST \
-d 'uname=Dummy&passwd=FUZZ&submit=Submit' \
-H 'Content-Type: application/x-www-form-urlencoded' --hc 1435 \
http://<IP>/<SUBDIRECTORY>/

Subdomain enumeration

wfuzz -c -w <WORDLIST> --hc 0 http://FUZZ.mydomain.com

Virtual hosting

wfuzz -c -w <WORDLIST> --hc 0 -H 'Host: FUZZ.mydomain.com' http://mydomain.com

Virtual hosting with IP

wfuzz -c -w <WORDLIST> -H 'Host: FUZZ.thm' http://<IP>
